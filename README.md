# 毕业设计----Python Web 个人博客设计
周四/2017/4/13 08:42:37 

----------

1. Web框架：[Django](https://www.djangoproject.com/)
2. 前端框架：[amazeui](http://amazeui.org/)
3. 内置编辑器（Django admin）：[Dracdeitor](https://github.com/agusmakmun/dracos-markdown-editor)

目前已经实现的功能：  
- 首页界面  
- 分类界面以及单个分类的文章列表  
- 文章、分类、标签的编辑显示（admin）  
- 关于博客界面  
- 文章详细界面
